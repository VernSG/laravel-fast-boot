# Extension Icon

To add an icon to the extension:

1. Create a 128x128 PNG icon
2. Save it as `icon.png` in this directory
3. Update `package.json` to reference it

The icon should:
- Be 128x128 pixels
- Have a transparent or white background
- Represent Laravel or fast bootstrapping
- Be visually appealing at small sizes
