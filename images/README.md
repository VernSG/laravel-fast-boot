1. Create a 128x128 PNG icon
2. Save it as `icon.png` in this directory
3. Icon should be 128x128 pixels with transparent or white background

## Demo/Screenshot

To add a demo for README:

**Option 1: GIF Animation**
1. Record your screen showing the extension in action
2. Convert to GIF using tools like:
   - [ScreenToGif](https://www.screentogif.com/) (Windows)
   - [Kap](https://getkap.co/) (macOS)
   - [Peek](https://github.com/phw/peek) (Linux)
3. Save as `demo.gif` (recommended: max 800px width, under 10MB)

**Option 2: Screenshot**
1. Take screenshot of the extension wizard
2. Save as `screenshot.png` or `demo.png`

**Option 3: Multiple Screenshots**
Create screenshots showing:
- Command palette entry
- Status bar button
- Setup wizard steps
- Completed setup

Name them: `screenshot-1.png`, `screenshot-2.png`, etc.
